# Source Directory Structure

This directory contains all C++ source files organized by functionality:

## Core (`core/`)
- **arbitrage.cpp** - Core arbitrage logic implementation
- **blockchain.cpp** - Blockchain interaction implementation
- **profit_analyzer.cpp** - Profit calculation implementation

## DEX (`dex/`)
- **dex_pools.cpp** - DEX pool management implementation
- **dex_scanner.cpp** - DEX scanning implementation
- **dex_stats.cpp** - DEX statistics implementation
- **dex_tokens.cpp** - Token management implementation

## CLI (`cli/`)
- **command_parser.cpp** - Command line parsing implementation
- **help_display.cpp** - Help display implementation

## Network (`network/`)
- **rpc_core.cpp** - RPC client implementation
- **queries.cpp** - Network queries implementation

## Utils (`utils/`)
- **measure.cpp** - Performance measurement implementation
- **input.cpp** - Input processing implementation
- **wallet.cpp** - Wallet management implementation

## Config (`config/`)
- **config_manager.cpp** - Configuration management implementation

## DI (`di/`)
- **service_container.cpp** - Dependency injection implementation

## Main
- **main.cpp** - Main application entry point
- **application.cpp** - Application core implementation
