# Utility Scripts

This directory contains utility scripts and tools for the DEXArb project.

## Scripts

### Neozork Tools
- `neozork-config` - Neozork configuration utility
- `neozork-scan-stat` - Neozork scan statistics tool

### Wine-Docker Utilities
- `wine-docker.sh` - Wine-Docker integration script

### Blockchain Utilities
- `scan-fantom.sh` - Fantom blockchain scanning utility

## Usage

### Neozork Configuration
```bash
./scripts/utilities/neozork-config
```

### Neozork Scan Statistics
```bash
./scripts/utilities/neozork-scan-stat
```

### Wine-Docker Integration
```bash
./scripts/utilities/wine-docker.sh
```

### Fantom Scanning
```bash
./scripts/utilities/scan-fantom.sh
```

## Utility Features

- Configuration management
- Blockchain data processing
- Wine-Docker integration
- Scan statistics and monitoring

## Dependencies

- Neozork server access
- Wine environment (for wine-docker.sh)
- Blockchain node access (for scan-fantom.sh)

## Notes

- Run from project root directory
- Ensure Neozork server is accessible
- Check Wine environment for wine-docker.sh
- Verify blockchain node connectivity for scanning tools
