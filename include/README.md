# Include Directory Structure

This directory contains all header files organized by functionality:

## Core (`core/`)
- **arbitrage.h** - Core arbitrage logic and algorithms
- **blockchain.h** - Blockchain interaction and data structures
- **profit_analyzer.h** - Profit calculation and analysis

## DEX (`dex/`)
- **dex_pools.h** - DEX pool management and operations
- **dex_scanner.h** - DEX scanning and discovery
- **dex_stats.h** - DEX statistics and metrics
- **dex_tokens.h** - Token management and operations

## CLI (`cli/`)
- **command_parser.h** - Command line argument parsing
- **help_display.h** - Help and usage information display

## Network (`network/`)
- **rpc_core.h** - RPC client implementation
- **queries.h** - Network query definitions

## Utils (`utils/`)
- **measure.h** - Performance measurement utilities
- **modern_format.h** - Modern data formatting
- **modern_result.h** - Result handling and formatting
- **input.h** - Input processing utilities
- **wallet.h** - Wallet management

## Config (`config/`)
- **config_manager.h** - Configuration management

## DI (`di/`)
- **service_container.h** - Dependency injection container

## Main
- **main.h** - Main application entry point
- **application.h** - Application core class
